Tools and Technologies

About software, the project application will be launched in 4 platforms such as Windows, MacOS, iOS and Android with some requirements of version. For example, the project demands Windows 7 or higher for Windows, at least iOS 10.0, above macOS Sierra 10.12 and for Android version 7.0 or higher will be allowed, and the team will require licenses for the aforementioned dev kits. Furthermore, the software team will use Microsoft Azure AI Platform for multitasking such as Machine Learning, Vision Capabilities and Azure ML which is a Python-based service. We will use a product of a cloud computing giant - Google, the product which is Google Cloud AU Platform will be used to store data and create portable ML pipelines. Then about hardware, we will need computers or laptops which will reach requirements about graphic cards for every member in this team. Modern cameras with computer vision attached are demanded for testing and improving the program. Some simple materials such as bell traps for animals, green LED, sensors are essential.

Testing

For the application, constant in-house testing of the features and of the AI is paramount. Hiring third-party QA agents is also up for consideration if the workload is too much for the in-house team. For the sensors, stress-testing for durability can be done by the hardware team, and the accuracy of the sensors can be tested on one of a team member’s pet (Khoi’s cat) with the help of a veterinarian.

Group Processes and Communication

Communication and communication skills have been an essential part in every group project, through it group members are able to deliver and receive information, ideas, changes, plans and updates for the project. In my group, we communicate face-to-face and online, we usually meet on RMIT campus after some lectures if we need to discuss something about the project, then we also make phone calls and have a group chat to discuss some minor issues daily. We have agreed that meetings take place twice a week and more if there is any emergency case. Normally, the meetings are held at RMIT campus after finishing lectures or tutorials and Facebook is the main platform that we believe to use for online communication and meetings. At first, our group will have a rule that members will be replaced if they are not active for three consecutive days without any acceptable reason and members have to tell the group if they have outside problems. We work on shared folders and documents such as Google Docs, Google Drive or Github, so that we can manage workload and processes of other members in the group. By that way, the new group members will have data, information and be able to continue on old members track if there is any disappeared teammate.

Skills and Jobs

AI software developer is the position that requires abilities of working on machine learning algorithms, using deep learning for business insights. They must be able to understand programming, software engineering, process data and develop AI systems. Have knowledge of many programming languages such as Python, Java or C/C++ and at least 2 years of experience in related fields. Communication skills are compulsory for this role in a project group for exchange ideas and information.

A data scientist is required to be good at several aspects such as Big Data, Maths, programming and statistical analysis. To become a group member as a data scientist, they have to achieve at least a Master's Degree in computer science or related field in order to have a strong foundation to solve technical and core problems. Furthermore, business acumen is a need to predict and bring more profitable value to the result in the future. Also, experience is required for this position as well as communication skills.

User experience designer which is usually called UX designer. Firstly, programming languages need to be mastered: HTML, CSS, Javascript and jQuery will help them remove transition between design handoff. User empathy plays an important role in success of the project because most customers do not care about software of the project, but they care about the output, the interface and how simple and easy to use the application. So that UX designers have to understand their problems, put themselves in users to know how they feel and have revolutionary changes. Moreover, visual communication is a must for UX designers because it has the most impact on users by interactive prototypes and vision. 

An animal healthcare professional is an essential part of the team and project because this is the project for animals and used on animals. We require to have a certificate of Animal Health and have experience at least five years on working with household pets, especially dogs and cats, completed daily animal care and have at least a well-presented report about animal habits, behaviours, health or any related aspect. Furthermore, professionals need to know how to use some basic computer tools such as Microsoft Offices, communication platforms, and knowledge of the IT field will be recommended.
